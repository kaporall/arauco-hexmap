# 🗺️ Arauco Hex Map — Mapa Isométrico Interactivo

**Live demo:** https://kaporall.github.io/arauco-hexmap

## Descripción
Mapa interactivo del Departamento Arauco, La Rioja, Argentina.
Grid hexagonal isométrico con 15 ubicaciones turísticas, históricas y culturales.
Construido con HTML5 Canvas + JavaScript puro, sin librerías externas.

## Tecnologías
- HTML5 Canvas (rendering 2D)
- JavaScript ES6+ (sin frameworks)
- CSS Variables + Animations
- Algoritmo axial hex grid

## Características
- 15 ubicaciones clickeables con info detallada
- Filtros por categoría (Naturaleza, Historia, Aventura...)
- Animaciones: lift effect, pulse ring, starfield
- Panel lateral con stats y descripciones
- Responsive (desktop y mobile)
- Tema oscuro premium
